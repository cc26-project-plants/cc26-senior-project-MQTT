## Getting Started: MQTT
1. clone this repository  
```bash
$ git clone https://github.com/cc26-project-plants/cc26-senior-project-MQTT.git
```  
2. move to the root directory  
```bash
$ cd cc26-senior-project-MQTT
```
3. download dependencies  
```bash
$ npm install
```
4. run the server
```bash
# development
$ npm start

# nodemon mode
$ npm run server
```
